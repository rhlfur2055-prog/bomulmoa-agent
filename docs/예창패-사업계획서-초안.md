# 예비창업패키지 사업계획서 (초안)

**아이템명(가칭): 보물모아 — 데이터로 물량을 모으는 자원순환 고물상 플랫폼**

---

## 0. 개요 / 아이템 한 줄 정의

**"동네 소매 고물상을 로컬 검색·실거래 시세 데이터·정기수거 운영으로 디지털화하여, 흩어진 자원 물량을 한곳에 집적(集積)하고 납품 단계를 상향시켜 마진을 개선하는 자원순환 사업."**

- 소프트웨어(장부 프로그램)를 파는 회사가 아니라, **실제로 고물상을 운영하며 물량을 모으는 사업**입니다. 기술은 물량을 모으기 위한 **차별화 수단(enabler)** 입니다.
- 이미 인천·부천 일대에서 가족이 고물상 업장을 운영 중이며(가동 중 4~5개소), 대표(창업자)는 개발자로서 현장에 부족했던 **온라인 유입·데이터·자동화** 역량을 직접 구축했습니다.

| 항목 | 내용 |
|---|---|
| 대표자 | ★사장님 확인 (창업자 = 개발자, 아들) |
| 사업 형태 | 가족 사업 (현장 운영: 부모님 / 기술·온라인: 대표) |
| 가동 업장 | 대림자원(부평), 모두다자원(인천), 대성자원(부천) 외 — 총 4~5개소 |
| 소재지 | 인천·부천 (경인권) |
| 취급 품목 | 고철(철스크랩)·비철금속 등 재활용 자원 |
| 현재 취급량/매출 | ★사장님 확인 |

---

## 1. 문제 인식 (Problem)

### 1-1. 소매 고물상의 구조적 한계

업계는 명확한 **단계 사다리**로 이루어져 있고, 위 단계로 갈수록 납품 단가가 오릅니다.

> 개인·소기업 → **소매 고물상** → 중상(집하·선별) → 대상(대량 납품) → 제강사

- **물량 부족 → 낮은 납품 단가**: 소매 단계는 물량이 적어 중상/대상에 낮은 단가로 넘길 수밖에 없습니다. **물량이 모일수록 납품 단가가 오르는 것이 마진 개선의 본질**입니다.
- **무기록·수기 운영**: 매입·매출·시세가 장부나 머릿속에 흩어져 있어, 어떤 품목을 어느 단가에 사고팔아야 유리한지 데이터 기반 의사결정이 어렵습니다.
- **온라인 부재 → 신규 유입 없음**: 대다수 소매 고물상은 온라인 존재감이 없어, "오늘 고철 얼마?" "방문수거 되나?"를 검색하는 신규 매입 고객을 잡지 못합니다.
- **장물(과실 취득) 리스크**: 신원 확인 없이 매입할 경우 과실로 장물을 취득할 위험이 있어, 사업 지속성과 신뢰에 부담이 됩니다.

### 1-2. 규제·제도 배경 (확인된 사실)

- **신원확인 법적 의무 공백**: 과거 고물영업법(1993년 폐지)이 사라지며 매입 시 신원확인의 법적 의무가 공백 상태입니다. 이 때문에 현재 업계의 실질적 구매 동기는 ① **과실 장물 취득 리스크 방어**, ② **세무(철스크랩 부가가치세 매입자 납부 특례 대응)** 입니다.
- **폐기물 재활용 신고 면적 기준**: 광역시 1,000㎡ / 시·군 2,000㎡. 사업 확장 시 **법인 전환 + 폐기물처리 신고**가 관공서·대기업 입찰 진입의 요건이 됩니다.

### 1-3. 시장 규모·성장성

- 국내 철스크랩·재활용 자원 시장의 정확한 규모/성장률: ★사장님 확인(공신력 있는 통계 인용 필요 — 제출 시 출처 명기).
- 확인 가능한 정성적 근거: 자원순환·재활용 정책 강화 흐름, 사업장폐기물 SaaS의 급성장(아래 3-1, 리코/업박스 사례) 등으로 **디지털화·집적 수요는 실재**합니다. 정량 수치는 근거 확보 후 기입.

---

## 2. 실현 방안 (Solution)

핵심은 **"물량을 모아 납품 단계를 올린다"** 이며, 이를 위해 네 가지 수단을 결합합니다. 상당 부분은 **이미 구축·가동 중**이라는 점이 예창패 심사의 '실증' 강점입니다.

### (a) 로컬 검색 장악 → 신규 매입 유입
- **로컬 SEO 사이트**(오늘의 단가·방문수거·거점 안내)를 이미 구축. "지역명 + 고철/비철 시세/방문수거" 검색 수요를 흡수하여 신규 매입 물량을 유입.
- (참고: 상세 부지·로컬검색 실행 문서는 별도 에이전트 담당 범위 — 본 계획서에서는 개념만 인용)

### (b) 정기수거 계약 → 안정적 물량 확보
- 사업장·상가·공장 등과 **정기수거 계약**을 맺어 산발적 매입을 넘어선 예측 가능한 물량 파이프라인을 확보.

### (c) 실거래 시세·단가 데이터 → 매입/납품 의사결정
- **고철/비철 시세 자동 수집 → 시트 → 사이트 자동 발행 파이프라인**을 이미 구축.
- 여기서 쌓이는 **지역 실거래 단가 DB**는, 경쟁사가 제공하는 전국 '참고 시세'에는 없는 고유 자산으로, 언제·무엇을·얼마에 사고팔지 결정하는 근거가 됩니다.

### (d) 물량 집적 → 납품 단계 상승
- 위 (a)~(c)로 모인 물량을 여러 업장에서 집적(集積)하여, 소매 → 중상 → 대상으로 **납품 단계 자체를 끌어올려** 단가·마진을 개선.

## 3. 성장 전략 (Scale-up)

### 3-1. 시장 경쟁 지도 요약 (상세는 부록)
국내에 "고물상 매입 컴플라이언스 장부"를 정면 타깃하는 사업자는 없음(틈새 확인). 인접 플레이어들은 각자 다른 조각을 검증해 줍니다(리코/업박스의 SaaS 성장, 민팃의 '신원확인+매입 자동화=사업' 국내 선례, 미국 컴플라이언스 SW 산업 등 — 부록 표 참조). 우리는 이들과 달리 **직접 물량을 모으는 오프라인 사업**을 데이터로 강화하는 각도입니다.

### 3-2. ROADMAP — 4단계 사업화 마일스톤

| 단계 | 목표 | 핵심 활동 | 상태 |
|---|---|---|---|
| **1. 지역 거점 장악** | 인천·부천 로컬 검색·매입 유입 1위 | 로컬 SEO 사이트, 방문수거, 실거래 단가 노출 | **진행 중(자산 구축 완료)** |
| **2. 전산화·지점 확장** | 매입/매출/시세 전산화, 업장 표준화·추가 | 데이터 파이프라인 현장 적용, 지점 운영 매뉴얼화 | 일부 착수 |
| **3. 물량 집적 파트너 네트워크** | 인근 소매 고물상·사업장을 파트너로 집적 | 정기수거 계약 확대, 파트너 납품 통합 | 계획 |
| **4. 자원순환 기업** | 법인 전환·폐기물처리 신고로 입찰 자격 확보, 대량 납품 | 법인화, 신고 면적 요건 충족, 관공서·대기업 납품 | 계획 |

### 3-3. 자금 사용 계획 (항목만 — 금액 ★사장님 확인)
- 집하장 부지 확보/임차 — ★금액 확인 (폐기물 신고 면적 기준 1,000/2,000㎡ 고려)
- 수거·운반 차량 — ★금액 확인
- 현장·운영 인력 — ★금액 확인
- 법인 전환 및 폐기물처리 신고 관련 비용 — ★금액 확인
- 사이트·데이터 파이프라인 운영/고도화 — ★금액 확인

> 금액·비중은 예창패 당해년도 공고의 사업비 집행 기준(인건비/외주/재료비 등 항목 구분)에 맞춰 재편성 필요.

## 4. 팀 (Team)

**기술 × 현장의 결합이 차별점입니다.**

| 역할 | 담당 | 강점 |
|---|---|---|
| 대표(창업자) | 아들, 개발자 | 로컬 SEO 사이트·시세 수집 자동화 파이프라인·데이터 DB를 직접 구축한 기술 역량 |
| 현장 운영 | 부모님 | 인천·부천 고물상 4~5개소 실운영 노하우, 업계 네트워크, 매입/납품 실무 |

- 대다수 경쟁 소매 고물상에는 없는 **개발 역량**과, 대다수 IT 스타트업에는 없는 **현장 실운영 자산**을 동시에 보유.
- 세부 이력·자격·역할 분담: ★사장님 확인.

## 5. 실증 / 트랙션 (Traction)

예창패 심사의 핵심인 **실증**에서 강점을 가집니다 — 아이디어가 아니라 **이미 돌아가는 사업**입니다.

- **가동 중인 업장**: 대림자원(부평)·모두다자원(인천)·대성자원(부천) 외 4~5개소.
- **구축된 기술 자산**:
  - 로컬 SEO 사이트(오늘 단가·방문수거·거점)
  - 고철/비철 시세 자동 수집 → 시트 → 사이트 자동 발행 파이프라인
  - 지역 실거래 단가 DB(경쟁사 참고시세에 없는 고유 자산)
- **구체 트랙션 수치**(월 취급량·매출·매입 건수·사이트 유입/전환 등): ★사장님 확인 — 제출 시 실데이터 기입.

## 부록 A. 시장 경쟁 지도

| 구분 | 플레이어 | 무엇을 하나 | 시사점 |
|---|---|---|---|

| 시세 정보(구독) | 스틸데일리 / ScrapMonster | 철스크랩 시세 구독(스틸데일리 **연 42~50만 원**) | 시세 정보에 지불 의사 존재 |
| 컴플라이언스 SW(해외) | ScrapRight / ScrapWare | 미국은 **주법이 신분증 캡처를 의무화** → 컴플라이언스 SW 산업 형성 | 모델의 해외 검증. 단, 한국은 법적 의무 공백이라 그대로 이식 불가 |

**전략적 위치**: 국내에 "고물상 매입 컴플라이언스 장부"를 정면 타깃하는 곳은 없음(틈새). 그러나 그 자체를 '제품'으로 파는 것이 아니라, 우리는 **직접 물량을 모으는 고물상 사업**을 데이터·자동화로 강화하는 각도로 진입합니다. 컴플라이언스/데이터는 물량 집적을 돕는 부수 수단입니다.
