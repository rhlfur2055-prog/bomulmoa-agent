# 부지 조사 — 인천 부평구 야적 부지(고철·비철 집하장) 신규 거점 딥다이브

> 목적: 기존 거점 **대림자원(부평 소재)** 반경 확장을 위한 부평구 내 신규 야적/집하장 후보 발굴.
> 방법: WebSearch(서버사이드) 기반 데스크 리서치. 아래 내용 중 **구체 매물의 주소·가격·면적은 전부 "미확인 — 직접 검증"** 이며, 확정 정보가 아니다.
> 작성일: 2026-07-12

---

## 0. 조사 원칙(중요)

- **지어내기 금지**: 확인 못 한 매물 주소/가격/면적은 절대 확정 서술하지 않는다. 단서만 있는 것은 반드시 **"미확인 후보"** 로 표기하고 원문 확인을 명시했다.
- 웹검색은 지역 개황·용도지역·정비구역·검색 경로 파악용이다. **개별 필지 판단은 토지이음(eum.go.kr) 지번 조회 + 현지 중개소 확인이 최종 근거**다.
- 부평구는 **인천광역시(광역시)** 소속이므로 폐기물 재활용 신고·용도지역 기준을 광역시 기준으로 적용한다.

## 1. 부평구 공업지대 지리 정리

### 1-1. 큰 그림

부평구의 공업 기능은 **청천동·갈산동·산곡동 일대(부평국가산업단지, 통칭 부평공단)** 를 축으로 형성돼 있고, 동쪽 **삼산동** 일대에 준공업·녹지가 혼재한 도농복합형 토지가 남아 있다. 나머지 부평동·부개동·십정동·일신동 등은 주거·상업 중심이며 대규모 재개발이 진행 중이라 야적 부지로는 부적합하거나 불안정하다.

### 1-2. 구역별 코멘트

| 구역 | 성격(확인 범위) | 야적 부지 적합성 | 리스크/경고 |
|---|---|---|---|
| **청천동(부평국가산업단지)** | 1969년 조성된 국내 최초 산업단지(한국수출산업단지 4단지), 청천동 일원 약 60만9,300㎡. 현재 **일반공업지역**, 준공업지역으로 용도변경 검토 이력(2019 경인일보). | 용도지역은 적합하나 **산업단지(계획입지)** 라 입주업종 제한·관리기관(한국산업단지공단) 승인 필요. 순수 야적/집하보다 제조업 위주. | 데이터센터(청천동, DL이앤씨)·지식산업센터 개발로 **부지 고도화·지가 상승 압력**. 단지 내 필지는 입주자격·업종제한 확인 필수. |

### 1-3. 핵심 시사점

- 순수 "**일반공업/준공업 나대지에서의 야적·집하**"만 놓고 보면 부평은 **① 청천동 부평공단(계획입지·업종제한), ② 삼산동 준공업 잔여지(정비구역 리스크), ③ 갈산동 일부(재개발 인접 주의)** 세 갈래로 좁혀진다.
- 부평 전역이 재개발·산단 고도화로 **공업 나대지가 빠르게 줄고 지가가 오르는 추세** → 매물 희소·가격 상승을 전제로 접근해야 한다.

## 2. 적격 필터 재확인 (부평 = 인천광역시 기준)

### 2-1. 최소 요건 체크리스트

1. **면적**: 폐기물 재활용 관련 신고/허가 기준상 **약 1,000㎡ 이상** 부지를 1차 컷으로 둔다.
   - ※ 정확한 신고·허가 대상 규모·시설기준은 **「폐기물관리법 시행규칙 별표」와 인천광역시/부평구 담당 부서 유권해석**으로 확정할 것(미확인 — 직접 검증). 취급 품목(고철·비철 = 지정외 재활용 대상 여부)과 보관량에 따라 신고/허가 구분이 달라진다.
2. **용도지역**: **일반공업·준공업·계획관리**만 허용. 그 외(주거·상업·녹지·자연/보전관리 등)는 컷.
   - 부평은 광역시라 계획관리지역은 사실상 삼산동 외곽 등 일부에 제한적. 실무상 **일반공업·준공업**이 주력.
3. **주거 이격**: 부지경계 **300m 내 주거(주택·아파트·학교)** 존재 시 **민원 리스크**로 감점/보류. 산곡동·부평동·부개동 인접지는 이 필터에서 대부분 탈락.
4. **정비구역/도시개발구역 지정 여부**: 지정·예정 구역은 **원천 제외**(갈산동 64-12, 삼산4지구 등).

### 2-2. 토지이음(eum.go.kr) 지번 조회 — 단계별

1. **접속**: `https://www.eum.go.kr` (회원가입 없이 무료 열람 가능).
2. **주소 입력**: 상단 통합검색창에 대상 필지의 **지번 주소**(예: "인천광역시 부평구 삼산동 000-0") 또는 도로명 주소 입력.
3. **필지 선택**: 검색 결과 또는 이음지도에서 해당 필지 클릭.
4. **핵심 확인 항목**:
   - **용도지역**(일반공업/준공업/계획관리 여부) — 이음지도 색상 구분으로도 확인.
   - **지목**(공장용지·잡종지·전·답 등 → 야적 시 지목·형질변경 필요 여부).
   - **면적**(1,000㎡ 이상 필터).
   - **행위제한 내용**(적치·야적·재활용시설 관련 제한).
   - **지역·지구·구역 지정**: **정비구역/도시개발구역/지구단위계획** 지정 여부 반드시 확인.
5. **고시정보 교차확인**: 토지이음 "고시정보"에서 해당 동·구역의 정비/도시개발 고시가 있는지 검색(예: "부평2 재개발", "삼산4지구", "갈산동 64-12").
6. **앱 활용**: 토지이음 앱(안드로이드/iOS)으로 현장에서 즉시 필지 규제·확인도면 열람 가능 → 현장 실사와 병행.

## 3. 실제 매물 탐색 딥다이브 (검색 경로)

> 아래는 **어떻게 좁히는가(경로)** 를 정리한 것이다. 개별 매물의 주소·가격은 **모두 미확인**이며 원문 로그인/열람으로 직접 확인해야 한다.

### 3-1. 인천산업부동산 (032-555-4555.com)

- 사이트: `https://032-555-4555.com` — **인천 공장·창고 임대/매매 전문**, "공장용지/나대지" 카테고리 존재(검색으로 카테고리·매물 리스트 페이지 확인됨).
- 경로: 상단 카테고리에서 **"공장용지/나대지"** 선택 → 지역 필터 **부평구** → 면적(㎡)·매매/임대·금액 범위로 좁힘.
- 활용 팁: 리스트가 지역 혼재라 **부평/부평구/청천동/삼산동/갈산동** 키워드로 재검색. 개별 매물은 **용도지역·지목이 사이트 표기와 다를 수 있으니 토지이음 교차확인**.
- **미확인**: 사이트 내 부평 나대지/공장용지 개별 매물의 실제 주소·가격·면적은 원문 로그인/문의로 직접 검증(직접 검증).

### 3-2. 디스코 (disco.re)

- 사이트: `https://disco.re` — **토지·빌딩·상가 실거래/매물 지도 플랫폼**.
- 경로: 지도에서 **부평구 청천동/삼산동/갈산동** 이동 → 필터에서 **토지(나대지·잡종지·공장용지)** 유형 선택, **면적·용도지역** 조건 지정 → 매물/실거래가 동시 확인.
- 활용 팁: 디스코는 **필지 클릭 시 용도지역·실거래가·토지이용계획을 함께 보여줘** 1차 스크리닝에 유리. 단, 매물 정보는 중개 등록분이라 최신성/정확성은 중개소 확인 필요.
- **미확인**: 디스코 상 부평 야적 가능 나대지 개별 매물은 직접 검증.

### 3-3. 네이버부동산

- 경로: 네이버부동산 → 지역 **인천 부평구** → 매물종류 **토지(공장용지/잡종지) 또는 공장·창고** → 키워드 검색창에 **"야적", "나대지", "공장용지", "적치"** 병행.
- 활용 팁: 야적장은 "토지" 또는 "공장/창고" 양쪽에 걸쳐 등록되므로 **두 카테고리를 모두** 훑어야 누락이 적다.

### 3-4. 보조 채널

- **114공장 (114gongjang.com)**: 인천 공장 매매/임대 리스트 → 부평 필터 + 부지(토지) 매물 확인.
- **부동산뱅크/네오넷 (neonet.co.kr)**: 부평구 토지/빌딩 매물 리스트.
- **KB부동산·밸류맵**: 실거래가·시세 교차확인용.
- **현지 산업부동산 중개소 직접 컨택**: 공단 인근 오프라인 중개소가 온라인 미노출 물건(나대지·야적)을 보유한 경우가 많음 → **실질 소싱은 오프라인 중개소가 핵심**.

> ⚠️ 이 절에서 언급한 사이트들에서 부평 야적 부지로 보이는 **개별 매물의 주소·가격·면적은 어느 것도 본 조사에서 확정 확인하지 못했다. 전부 "미확인 후보"이며 원문 열람·중개소 확인이 필수다.**

## 4. 추천 후보 동(洞) 3곳 — 우선순위

> "여기부터 토지이음 + 중개소 확인" 성격의 **탐색 우선순위**이지 매물 확정이 아니다.

### 1순위 — 삼산동 (준공업 잔여지)
- **이유**: 준공업·녹지 혼재 도농복합으로 **부평 내에서 상대적으로 큰 필지·나대지가 남아있을 가능성**이 가장 높음. 서울외곽순환/경인 물류 접근 양호.
- **단, 강한 경고**: **삼산4지구(삼산동 325 일대) 도시개발구역 추진 구역**과 겹치면 부지 불안정. 또한 부평구가 **삼산동 국·공유지 불법 야적 행정대집행**으로 단속 강화 중 → **정비구역 밖 + 사유지 + 합법 적치 요건**을 반드시 확인. 삼산4지구 경계 확인이 최우선.

### 2순위 — 청천동 (부평국가산업단지)
- **이유**: 용도지역(일반공업, 준공업 전환 검토)·인프라·도로가 가장 정비돼 있음. 기존 거점 대림자원과 **동일 공업 축**이라 반경 확장 취지에 부합.
- **단, 경고**: **계획입지(산업단지)** 라 입주업종·관리기관 승인 제약이 있고, 데이터센터·지식산업센터 개발로 **지가 상승·부지 고도화** 진행 중. 순수 야적보다 제조 겸업 형태가 현실적. 단지 내 필지는 입주자격 확인 필수.

### 3순위 — 갈산동 (공단 축 일부)
- **이유**: 부평공단 축의 일부로 공업·준공업 필지 존재, 도로 접근 양호.
- **단, 경고**: **갈산동 64-12 일대 재개발(1,727세대) 추진** — 해당·인접 구역은 배제. 재개발 경계 밖의 공업 필지만 선별 검토.

> **제외 권고**: 산곡동(재개발 완료·주거 전환), 부평동·부개동·십정동·일신동(주거/상업·재개발 진행)은 야적 부지 후보에서 제외.

## 5. 현장 실사 체크리스트

후보지 방문 시 확인 항목:

- [ ] **진출입 도로폭**: 25톤 이상 대형 트럭·트레일러 진입 가능한 도로폭·회전반경 확보 여부. 접도 상태(막다른 길·일방 여부).
- [ ] **도로 접면**: 필지가 도로에 실제 접해 있는지(맹지 여부), 진입로 소유관계(사도/공도).
- [ ] **전기**: 계량기·수전용량(중장비·자석크레인·전단기 사용 시 충분한지). 증설 가능성.
- [ ] **상하수도**: 급수·오수 연결 가능 여부, 정화조 유무.
- [ ] **우수·오수 분리**: 우수/오수 분리배관 여부, 침출수·세륜 처리 가능성(비철 세척·강우 유출 대비). 인접 하천·우수관 오염 리스크.
- [ ] **지목**: 공장용지/잡종지 여부, 야적 위해 형질변경·개발행위허가 필요한지.
- [ ] **용도지역 현장 대조**: 토지이음 조회값(일반공업/준공업/계획관리)과 실제 주변 이용현황 일치 여부.
- [ ] **정비구역/도시개발구역 여부**: 정비구역·지구단위계획·도시개발 예정 구역 포함 여부(토지이음 고시정보 + 구청 도시정비과 재확인). **지정 구역이면 배제.**
- [ ] **인접 민가·시설 거리**: 부지경계에서 **가장 가까운 주택·아파트·학교까지 실측 거리(300m 필터)**. 소음·분진·차량 민원 예상 지점 파악.
- [ ] **지반·배수·침수**: 지반 다짐 상태, 우천 시 침수 이력(저지대 여부), 야적 하중 견딤.
- [ ] **경계·담장·보안**: 필지 경계 명확성(경계측량 필요 여부), 담장·CCTV·야간 보안.
- [ ] **바닥 포장**: 콘크리트/아스콘 포장 여부(비포장 시 분진·침출수 관리 비용).
- [ ] **인허가 사전상담**: 부평구청 환경부서(폐기물)·건축/도시계획 부서에 **재활용 신고 대상 여부·시설기준·입지 가능성 사전 문의**.
- [ ] **주변 동종업 존재**: 인근에 고물상·재활용업체가 이미 있는지(민원 내성·집적 효과 판단).
