# Bomulmoa 시트 사이드바 작업관리 앱 🪙

구글 시트 **안에서** 바로 뜨는 사이드바 앱. Slack·서버·토큰 없이 시트만 있으면 됩니다.
팀원별 **오늘 작업 / 시간 / 어디까지 했는지 / 할당량 대비 진행률**을 기록하고, 버튼 하나로 **txt 문서화**합니다.

## 무엇을 해주나
- ⏱ **일일 작업 기록** — 팀원·트랙·작업내용·시간·"어디까지 했는지"·진행률(%)
- 📊 **실시간 팀 현황** — 하루 할당량(기본 6h) 대비 달성률 막대그래프
- 📄 **txt 내보내기** — 오늘/전체 진행상황을 Google Drive에 `.txt`로 저장
- 🗺 **1개월 로드맵** — 백엔드·프론트·AI 트랙 주차별 목표를 `프로젝트계획` 탭에 자동 생성

자동 생성 탭: `일일작업로그` · `할당량` · `프로젝트계획`
(할당량 탭에서 팀원/트랙/하루 할당시간을 자유롭게 수정하세요.)

## 설치 (5분)
1. 대상 시트 열기 → 상단 메뉴 **확장 프로그램 → Apps Script**
2. 기본 `Code.gs` 내용을 지우고 이 폴더의 **[Code.gs](Code.gs)** 붙여넣기
3. 왼쪽 **＋ → HTML** 로 파일 추가, 이름을 정확히 **`Sidebar`** 로 하고 **[Sidebar.html](Sidebar.html)** 붙여넣기
4. 저장(💾) → 함수 목록에서 **`setup`** 선택 → **▶ 실행** (첫 실행 시 권한 승인)
5. 시트로 돌아가 **새로고침** → 상단에 **🪙 Bomulmoa** 메뉴 생김
6. **🪙 Bomulmoa → 작업 사이드바 열기**

이제 오른쪽에 사이드바가 뜹니다 (스크린샷의 Gemini 패널과 같은 위치).

## 팀원별 사용법
- 각자 시트에서 사이드바 열고 → 본인 이름 선택 → 작업 기록하면 끝
- 하루 여러 번 기록하면 시간이 누적되고, 마지막 "어디까지"가 현황에 표시됨
- 퇴근 전 **📄 오늘 txt** 눌러 하루 진행상황 문서화

## 커스터마이징
- **하루 할당량 변경**: `할당량` 탭의 C열 숫자 수정
- **팀원 추가**: `할당량` 탭에 행 추가 (이름 / 트랙 / 할당h)
- **로드맵 수정**: `프로젝트계획` 탭 직접 편집
